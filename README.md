# Tamper-Evident Logging System

## Overview
This project implements a secure logging system where logs cannot be modified without detection using hash chaining.

## Features
- Secure log chaining using SHA-256
- Tamper detection
- Integrity verification

## How to Run
1. Install Python
2. Run:
   python main.py

## Output
- Displays logs
- Verifies integrity
- Detects tampering
